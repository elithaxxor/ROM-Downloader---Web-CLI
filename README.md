Welcome to **ROM Downloader**! This is your ultimate tool to automatically download ROMs either via a CLI or a web interface. Download single or multiple files effortlessly! 

## ✨ Features

- 🖥️ **CLI Interface**: Easy-to-use command-line interface for downloading ROMs.
- 🌐 **Web Interface**: User-friendly web interface for managing your downloads.
- 📂 **Batch Downloads**: Download multiple ROMs at once.
- 🔍 **Search Functionality**: Find and download your favorite ROMs quickly.
- 🛠️ **Customizable Settings**: Tailor the download settings to your needs.

## 🚀 Getting Started

### Prerequisites

Make sure you have Python installed on your system. This project also uses a few Python libraries which can be installed via `pip`.

### Installation

1. **Clone the repository**:

    ```bash
    git clone https://github.com/elithaxxor/rom_downloader.git
    cd rom_downloader
    ```

2. **Install dependencies**:

    ```bash
    pip install -r requirements.txt
    ```

### Usage

#### CLI Mode

Run the downloader in CLI mode:

```bash
python cli_downloader.py
```

#### Web Interface

Run the downloader in Web mode:

```bash
python web_downloader.py
```

Open your browser and navigate to `http://localhost:8000` to access the web interface.
